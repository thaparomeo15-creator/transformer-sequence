# TRANSFORMER SEQUENCE //

World-class scroll-driven canvas engine built with Next.js, GSAP, and Web Workers.

## 🚀 Pillars of Performance

- **Zero Latency**: Frame decoding happens off-thread in Web Workers.
- **Cinematic Fluidity**: Lenis smooth scroll + GSAP ScrollTrigger synchronization.
- **Glassmorphic UI**: High-end monospace aesthetic with obsidian-shell styling.
- **Secure Commerce**: Full Stripe & Supabase integration for licensing.

## 🛠️ Stack

- **Framework**: Next.js 15 (App Router)
- **Animation**: GSAP, Framer Motion
- **Scroll**: Lenis
- **Database**: Supabase
- **Payments**: Stripe
- **Email**: Resend

## 📦 Setup

1. **Install dependencies**: `npm install`
2. **Environment**: Copy `.env.example` to `.env.local` and fill in your keys.
3. **Frames**: Add 204 WebP frames to `/public/frames/001.webp` → `204.webp`.
4. **Dev**: `npm run dev`
